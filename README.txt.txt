Description:
This assignment was a lesson in list views and algorithms that would allow the user to check for duplicate list objects in an arraylist. The list views would use intents to navigate between classes and start new activities.
Team member roles in assignment: 
Harsha: Documentation
Gustavo: Program
Uday: Program
Megan: Documentation
Key Lessons learned:
The most important teaching point of the lesson was intents. Intents are extremely useful in a large application because they can start a new activity within a current activity. In addition, data can be added to activities and information can be pulled from activities.
